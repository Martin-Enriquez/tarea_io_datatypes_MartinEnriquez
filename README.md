Este programa permite convertir divisas entre USD, EUR y JPY utilizando una tasa de cambio ingresada por el usuario. 
El sistema ofrece dos modos de operación: modo continuo (conversiones individuales)
 y modo sesión (múltiples conversiones con estadísticas acumuladas). 
El programa incluye validación estricta de entradas, control de errores con `cin.fail()` y formateo de salida con
 dos decimales y separador de miles.

Compilar:

g++ deber1.cpp -o deber1

Ejecutar:

./deber1
